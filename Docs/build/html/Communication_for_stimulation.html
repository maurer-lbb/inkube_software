Communication_for_stimulation module¶

class Communication_for_stimulation.Communication(host, port, auth_key='1234')¶

Bases: object

Communication class for electrophysiology which can switch the mode of operation, send stimuli and receive electrophysiology data

Empty the spontaneous spike queue Returns:

int: Returns 1 if the queue is emptied

get_response()¶

Get the resposne when main is in stimulation mode Returns:

dict: Returns the response from the simulation

Get the resposne when main is in spontaneous mode Returns:

dict: Returns the spontaneous spikes from the simulation

new_connection()¶

Establishes a new connection with the specified host and port and registers the data exchange pipes and queues. Returns:

int: Returns -1 if the connection could not be established, otherwise returns None.
