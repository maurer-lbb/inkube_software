USB_communication module¶

class USB_communication.USB_com¶

Bases: object

Class for the USB communication from the host PC to the FPGA/SoC. This uses USB bulk communication on a single port.

close()¶

Close the USB communication.

prepare_preamble(command_key)¶

Prepare the preamble for the USB communication.

send_recv_port(port_id, recv_pkg_id, retry=1, retry_timeout=0.5)¶

Send a command to the USB device to set the port for USB communication.

write_data(data: bytearray)¶

send data to the USB device.

USB_communication.relay_fpga_commands_process(env_commands_pipe: Connection, command_to_send_pipe: Connection)¶

Process to receive register writes to the FPGA from the ControlPort Jupyter Client and send them to the USB communication process. Args:

env_commands_pipe (multiprocessing.connection.Connection): Pipe to receive commands from the ControlPort JupyterClient through a network socket. command_to_send_pipe (multiprocessing.connection.Connection): Pipe to send commands to the USB communication process.

USB_communication.send_commands_process_USB(com: USB_com, command_to_send_pipe: Connection, recv_pkg_id: Connection, retry_timeout=0.05)¶

Process to send commands to the USB device. Args:

com (USB_com): USB communication object. command_to_send_pipe (multiprocessing.connection.Connection): Pipe to receive commands to send to the USB device. recv_pkg_id (multiprocessing.connection.Connection): Pipe to receive the incrementing counter of received data on the SoC which makes sure no data is dropped. retry_timeout (float, optional): Timeout for the retry. Defaults to 500e-4.
