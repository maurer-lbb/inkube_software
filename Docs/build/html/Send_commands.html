Send_commands module¶

This script contains functions for sending commands to the intan RHS2116 and reading data from its registers.

Send_commands.get_command_pos_from_source(src: tuple = (0, 0), command=0)¶

Derive position in command frame for command depending on target mea and chip

Send_commands.prepare_commands_process(command_pipe: Connection, command_to_send_pipe: Connection, newest_recv_pkg)¶

Send stimulation commands when put on command pipe Args:

command_pipe: connection to receive stimulation commands command_to_send_pipe: connection to send commands to the USB process newest_recv_pkg: shared array to store the latest received package id from the UDP stream

Send_commands.read_from_registers(command_to_send_pipe: Connection, registers=[32, 33, 34, 35, 36, 37, 40, 42, 44, 46, 48, 64, 96], init_source=None)¶

Read data from registers on the intan chip, used for debugging Args:

command_to_send_pipe: connection to send commands to the USB process registers: list of registers to read from init_source: list of mea and chip to send commands to, if None send to all chips

Send_commands.send_write_to_register(command_to_send_pipe: Connection, command_list=[(32, [0, 0]), (33, [0, 0]), (1, [0, 0]), (4, [0, 22]), (5, [0, 23]), (6, [0, 10]), (7, [0, 18]), (8, [255, 255]), (38, [255, 255]), (10, [0, 0]), (12, [0, 0]), (34, [105, 65]), (35, [0, 102]), (36, [0, 128]), (37, [79, 0]), (42, [0, 0]), (44, [255, 255]), (46, [0, 0]), (48, [0, 0]), (64, [128, 2]), (65, [128, 2]), (66, [128, 2]), (67, [128, 2]), (68, [128, 2]), (69, [128, 2]), (70, [128, 2]), (71, [128, 2]), (72, [128, 2]), (73, [128, 2]), (74, [128, 2]), (75, [128, 2]), (76, [128, 2]), (77, [128, 2]), (78, [128, 2]), (79, [128, 2]), (96, [128, 2]), (97, [128, 2]), (98, [128, 2]), (99, [128, 2]), (100, [128, 2]), (101, [128, 2]), (102, [128, 2]), (103, [128, 2]), (104, [128, 2]), (105, [128, 2]), (106, [128, 2]), (107, [128, 2]), (108, [128, 2]), (109, [128, 2]), (110, [128, 2]), (111, [128, 2]), (32, [170, 170]), (33, [0, 255])], init_source=None, timed_send=False, pkg_id=2147483648)¶

Send commands to initialise stimulation units Args:

command_to_send_pipe: connection to send commands to the USB process command_list: list of commands to send init_source: list of mea and chip to send commands to timed_send: flag to send commands at specific timepoint pkg_id: package id to send commands at, if MSB is 1 send immediately

Send_commands.set_amplitude(value, command_list=[])¶

Set the amplitude of the stimulation pulse. The amplitude is set in steps according to settings in Client_config.py.

Send_commands.set_digaux(send_pipe, value=1, pkg_id=2147483648)¶

Set digital auxiliary output on or off Args:

send_pipe: connection to send commands to the USB process value: flag to set digital auxiliary output on or off pkg_id: package id to send commands at, if MSB is 1 send immediately

Send_commands.set_stimulus_timing_local(stim_duration, discharge_time)¶

Set the timing for the stimulation pulses Args:

stim_duration: duration of the stimulation pulse discharge_time: time to discharge the electrodes in packages

Send_commands.write_to_register(r, d)¶

get control word to write data to register on intan chip
